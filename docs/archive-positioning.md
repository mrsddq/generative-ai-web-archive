# Archive Positioning

This repository is a generative AI web learning archive.

## Keep

- Short README explaining what was learned.
- Any small demo screenshots or architecture notes.
- Honest limitations around API keys, model availability, and old dependencies.

## Avoid

- Claiming production readiness.
- Committing `.env`, virtual environments, generated caches, or model outputs.
- Presenting old experiments as current deployment work.

Best future move: rebuild the core idea as a clean, small app if it becomes a portfolio project again.
